Here’s a tiny gongfu-style timer with the steep getting a little longer each round:

```go
func Steep(infusion int) time.Duration {
	return time.Duration(20+infusion*5) * time.Second
}
```

Start around **95°C**, use plenty of leaf, and taste between infusions. A rolled Tieguanyin may open slowly; a roasted Wuyi oolong often gives up its aroma sooner.
